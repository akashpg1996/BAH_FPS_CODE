# Hello Python App

This is a simple Python application that prints "Hello, World!" to the console.

## Project Structure

```
BAH_FPS
├── src
│   └── BAH.py
    └── FPS.py 
    └── functions.py                
├── requirements.txt
└── README.md
```

## How to Run the Application

1. Make sure you have Python installed on your machine.
2. Navigate to the project directory.
3. Run the application using the following command:

   ```
   pip install -r requirements.txt
   pip install .
   ```

## Dependencies

This project does not have any external dependencies. The `requirements.txt` file is intentionally left blank for this basic application.
